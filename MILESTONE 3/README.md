Project Overview



Milestone 3 focuses on building an AI-powered Question \& Answer Multi-Language Engine with summarization capabilities and knowledge graph integration.



The system processes user queries, retrieves relevant information, summarizes content, and visualizes relationships between entities using a knowledge graph.



Knowledge graphs organize information as entities and relationships, enabling intelligent search and reasoning within the system.



This milestone also ensures full integration with the previously developed Milestone 2 features, including authentication, UI improvements, and analytics dashboards.



Milestone 3 Features

1\. Q\&A Multi-Language Engine



A question-answering engine capable of processing user queries in multiple languages.



Key functionalities:



Accepts questions in multiple languages



Performs translation when required



Retrieves relevant information



Generates accurate responses



2\. Multi-Language Summarization



The system summarizes large textual information into concise responses.



Features include:



Summarizing web content or documents



Multi-language support



Improved readability and concise output



3\. Knowledge Graph Integration



A knowledge graph was implemented to extract entities from text and map relationships between them.



Features:



Entity extraction



Relationship mapping



Visualization of connections between entities



Structured representation of information



Knowledge graphs structure data into interconnected entities, enabling better information retrieval and reasoning for AI systems.



Integration with Previous Milestones
