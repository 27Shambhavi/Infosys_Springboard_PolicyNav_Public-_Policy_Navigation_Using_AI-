# Infosys_Springboard_PolicyNav_Public _Policy_Navigation_Using_AI 
# Milestone 1 – User Authentication System

## Project Title

PolicyNav – Public Policy Navigation Using AI

## Description

In this milestone, a secure user authentication system was developed using **Streamlit, MySQL, JWT, and Ngrok**.
This system provides user signup, login, dashboard access, and password recovery functionality.

## Features Implemented

* User Signup with validation
* Secure Login using JWT
* Dashboard after login
* Forgot Password using security question
* Public URL generation using Ngrok

## Technologies Used

* Python
* Streamlit
* MySQL
* JWT Authentication
* Ngrok

## How to Run the Project

1. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```

2. Run Streamlit app

   ```bash
   streamlit run app.py
   ```

3. Start Ngrok

   ```bash
   ngrok http 8501
   ```

4. Open the HTTPS public URL shown in terminal.
